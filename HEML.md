## Installation
Install dependencies
```console
$ sudo apt install libntl*
$ sudo apt install libpthread-*
```

Get and install the HEAAN library from:
https://github.com/kimandrik/HEAAN
```console
$ git clone https://github.com/kimandrik/HEAAN.git
$ cd HEAAN/HEAAN/lib
$ make all
```
